# ACT_0016 - Visualizacion basica de resultado JSON en app Expo

## Objetivo tecnico
Mostrar en la app Expo el resultado de lectura recibido desde backend para validacion funcional MVP.

## Tareas implementables
- [ ] Renderizar resumen minimo (total preguntas, marcadas, ambiguas).
- [ ] Mostrar listado por pregunta con opcion marcada.
- [ ] Incluir vista de JSON crudo para inspeccion manual durante pruebas.
- [ ] Manejar estados de error/resultado vacio de forma clara.

## Evidencias esperadas
- Vista de resultados en app Expo con datos de una foto procesada.
- Flujo completo demostrable desde captura hasta visualizacion.

## Criterio de terminado
El usuario puede ver en el celular las respuestas detectadas por pregunta tras tomar y enviar una foto.
